MODULE 1 EXERCISES - BASIC FOUNDATIONS Use Quality Control doc

All task needs to be done by using prompt only.

Task 1: Summarize requirements
Question: How would you ask Copilot to summarize a user story in simple terms?
Answer: "Summarize this user story in 2-3 sentences focusing on main functionality"

Task 2: Identify risks
Question: What prompt helps find testing risks in a feature?
Answer: "What are the main testing risks in this feature?"

Task 3: Find ambiguities
Question: How would you prompt to find unclear requirements?
Answer: "What details are missing or unclear in these requirements?"

Task 4: Extract test scenarios
Question: What simple prompt generates basic test ideas?
Answer: "List 5 test scenarios for this functionality"

Task 5: Understand acceptance criteria
Question: How would you ask Copilot to explain acceptance criteria?
Answer: "Explain these acceptance criteria in simple terms"

Task 6: Find edge cases
Question: What prompt identifies edge cases from requirements?
Answer: "What are the edge cases for this feature?"

Task 7: Check completeness
Question: How would you verify if requirements are complete?
Answer: "Are there any missing requirements or gaps in this specification?"

Task 8: List assumptions
Question: What prompt finds assumptions in requirements?
Answer: "What assumptions are being made in these requirements?"

Task 9: Generate questions for clarification
Question: How would you create questions to ask stakeholders?
Answer: "Generate clarification questions for these requirements"

Task 10: Create simple test checklist
Question: What prompt makes a basic testing checklist?
Answer: "Create a simple test checklist for this feature"
